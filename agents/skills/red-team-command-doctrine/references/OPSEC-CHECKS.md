# OPSEC Checks

1. Is there one lower-noise path I can prove first?
2. Am I broadening scope without new evidence?
3. Will this action generate avoidable logs, auth events, or scans?
4. Can I use native evidence (Burp, CLI, host facts) instead of assumptions?
5. Is the next step directly tied to the objective?
6. Will this burn current access, credentials, or infrastructure?
7. Does the expected value justify the detection surface?
8. If interrupted now, is the current state recoverable?
